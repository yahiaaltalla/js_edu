PHP HOME

Welcome to the PHP tutorial. PHP is a popular scripting language for creating dynamic web pages.

PHP Intro

PHP stands for "PHP: Hypertext Preprocessor". It is a widely-used, open-source scripting language.

PHP Install

To install PHP, download it from the official PHP website and configure it on your server.

PHP Syntax

PHP scripts are executed on the server, and the output is returned to the browser as plain HTML. Example:

<?php
  echo "Hello, World!";
?>

PHP Comments

Comments are used to explain the code. Example:

// This is a single-line comment
# This is also a single-line comment
/* This is a 
multi-line comment */

PHP Variables

Variables are used to store data. Example:

<?php
  $name = "John";
  $age = 25;
  echo $name . " is " . $age . " years old.";
?>

PHP Echo / Print

Both echo and print are used to output data. Example:

<?php
  echo "Hello, World!";
  print "Hello, PHP!";
?>

PHP Data Types

PHP supports several data types, including strings, integers, and arrays. Example:

<?php
  $string = "Hello";
  $int = 42;
  $array = ["Apple", "Banana", "Cherry"];
?>

PHP Strings

Strings are sequences of characters. Example:

<?php
  $greeting = "Hello, " . "World!";
  echo $greeting;
?>

PHP Numbers

PHP supports integers and floats. Example:

<?php
  $x = 10;
  $y = 10.5;
?>

PHP Casting

Casting allows you to convert variables to different types. Example:

<?php
  $x = 10.5;
  $int_cast = (int)$x;
  echo $int_cast;
?>

PHP Math

PHP provides functions for mathematical operations. Example:

<?php
  echo sqrt(16); // 4
?>

PHP Constants

Constants are defined using the define() function. Example:

<?php
  define("SITE_NAME", "My Website");
  echo SITE_NAME;
?>

PHP Magic Constants

PHP provides predefined constants that change depending on their context. Example:

<?php
  echo __FILE__;
?>

PHP Operators

PHP supports various operators, such as arithmetic and logical operators. Example:

<?php
  $x = 5;
  $y = 10;
  echo $x + $y;
?>

PHP If...Else...Elseif

Conditional statements allow decision-making. Example:

<?php
  $age = 20;
  if ($age >= 18) {
      echo "Adult";
  } else {
      echo "Minor";
  }
?>

PHP Switch

The switch statement is used for multiple conditions. Example:

<?php
  $color = "red";
  switch ($color) {
      case "red":
          echo "Stop!";
          break;
      case "green":
          echo "Go!";
          break;
      default:
          echo "Wait!";
  }
?>

PHP Loops

Loops execute a block of code repeatedly. Example:

<?php
  for ($i = 0; $i < 5; $i++) {
      echo $i;
  }
?>

PHP Functions

Functions encapsulate code to perform specific tasks. Example:

<?php
  function sayHello() {
      echo "Hello, World!";
  }
  sayHello();
?>

PHP Arrays

Arrays store multiple values. Example:

<?php
  $fruits = ["Apple", "Banana", "Cherry"];
  foreach ($fruits as $fruit) {
      echo $fruit . " ";
  }
?>

PHP Superglobals

Superglobals are predefined variables in PHP. Example:

<?php
  echo $_SERVER['PHP_SELF'];
?>

PHP RegEx

Regular expressions are used for pattern matching. Example:

<?php
  if (preg_match("/world/", "Hello world!")) {
      echo "Match found.";
  }
?>

PHP Forms

PHP is used to handle form data. Example:

<form method="post" action="">
  Name: <input type="text" name="fname">
  <input type="submit">
</form>

PHP Form Handling

Handle form submissions securely. Example:

<?php
  if ($_SERVER["REQUEST_METHOD"] == "POST") {
      $name = htmlspecialchars($_POST['fname']);
      echo $name;
  }
?>

PHP Form Validation

Validate form input for better security. Example:

<?php
  if (empty($_POST['fname'])) {
      echo "Name is required.";
  }
?>

PHP Form Required

Mark form fields as required. Example:

<form method="post" action="">
  Name: <input type="text" name="fname" required>
  <input type="submit">
</form>

PHP Form URL/E-mail

Validate URLs and email addresses. Example:

<?php
  $email = filter_var($email, FILTER_VALIDATE_EMAIL);
  $url = filter_var($url, FILTER_VALIDATE_URL);
?>

PHP Form Complete

A complete form example:

<?php
  if ($_SERVER["REQUEST_METHOD"] == "POST") {
      $name = htmlspecialchars($_POST['fname']);
      echo "Thank you, " . $name;
  }
?>
<form method="post" action="">
  Name: <input type="text" name="fname" required>
  <input type="submit">
</form>

PHP Date and Time

PHP provides functions to work with dates and times. Example:

  <?php
    echo date("Y-m-d H:i:s");
  ?>

PHP Include

Use include to insert the content of one PHP file into another. Example:

  <?php
    include 'header.php';
  ?>

PHP File Handling

Handle files using functions like fopen, fread, and fwrite. Example:

  <?php
    $file = fopen("example.txt", "r");
    echo fread($file, filesize("example.txt"));
    fclose($file);
  ?>

PHP File Open/Read

Open and read files with fopen and fread. Example:

  <?php
    $file = fopen("example.txt", "r");
    while (!feof($file)) {
        echo fgets($file);
    }
    fclose($file);
  ?>

PHP File Create/Write

Create and write to files using fopen and fwrite. Example:

  <?php
    $file = fopen("newfile.txt", "w");
    fwrite($file, "Hello, World!");
    fclose($file);
  ?>

PHP File Upload

Upload files to the server. Example:

  <form action="upload.php" method="post" enctype="multipart/form-data">
    Select file: <input type="file" name="fileToUpload">
    <input type="submit" value="Upload File">
  </form>

PHP Cookies

Set and retrieve cookies. Example:

  <?php
    setcookie("user", "John", time() + (86400 * 30), "/");
    echo $_COOKIE["user"];
  ?>

PHP Sessions

Start a session to store user information. Example:

  <?php
    session_start();
    $_SESSION["user"] = "John";
    echo $_SESSION["user"];
  ?>

PHP Filters

Validate and sanitize input. Example:

  <?php
    $email = filter_var("test@example.com", FILTER_VALIDATE_EMAIL);
    echo $email;
  ?>

PHP Filters Advanced

Use advanced filters for custom validation. Example:

  <?php
    $options = ["options" => ["min_range" => 1, "max_range" => 100]];
    echo filter_var(50, FILTER_VALIDATE_INT, $options);
  ?>

PHP Callback Functions

Use callback functions in PHP. Example:

  <?php
    function my_callback($item) {
        return strtoupper($item);
    }
    $arr = ["apple", "banana", "cherry"];
    print_r(array_map("my_callback", $arr));
  ?>

PHP JSON

Work with JSON data in PHP. Example:

  <?php
    $arr = ["name" => "John", "age" => 30];
    echo json_encode($arr);
  ?>

PHP Exceptions

Handle exceptions in PHP. Example:

  <?php
    try {
        throw new Exception("An error occurred.");
    } catch (Exception $e) {
        echo $e->getMessage();
    }
  ?>

PHP What is OOP

Object-Oriented Programming (OOP) is a programming style that uses objects and classes.

PHP Classes/Objects

Classes are blueprints for objects, and objects are instances of classes.

  <?php
    class Car {
        public $color;
        public $model;
        public function __construct($color, $model) {
            $this->color = $color;
            $this->model = $model;
        }
    }
  ?>

PHP Constructor

Constructors are special methods that are automatically called when an object is created.

PHP Destructor

Destructors are special methods that are automatically called when an object is destroyed.

PHP Access Modifiers

Access modifiers define the visibility of class properties and methods.

PHP Inheritance

Inheritance allows a class to use properties and methods of another class.

PHP Constants

Constants are like variables but cannot be changed after they are set.

PHP Abstract Classes

Abstract classes cannot be instantiated and are meant to be extended by other classes.

PHP Interfaces

Interfaces define methods that must be implemented by a class.

PHP Traits

Traits are used to include reusable methods in multiple classes.

PHP Static Methods

Static methods can be called without creating an instance of a class.

PHP Static Properties

Static properties are shared by all instances of a class.

PHP Namespaces

Namespaces are used to organize code and avoid name conflicts.

PHP Iterables

Iterables are data structures that can be looped through with a foreach loop.
