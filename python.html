Python Intro

Python is a high-level, interpreted programming language used for web development, data science, artificial intelligence, and more. It is known for its readability and simplicity.

Python Syntax

Python syntax is clean and easy to read. Example of a basic Python program:

print("Hello, World!")

Python Variables

Variables are used to store data. Example:

x = 10
y = "Hello"

Python Data Types

Python has various data types such as integers, floats, strings, and booleans. Example:

age = 30        # Integer
height = 5.7    # Float
name = "Alice"  # String
is_active = True  # Boolean

Python Numbers

Python supports various numerical types such as int and float. Example:

x = 10
y = 15.5

Python Casting

Casting is used to convert one data type to another. Example:

x = 10
y = float(x)  # Convert to float

Python Strings

Strings are sequences of characters. Example:

name = "Alice"
greeting = "Hello, " + name

Python Booleans

Booleans represent True or False values. Example:

is_active = True
is_done = False

Python Operators

Python supports various operators like arithmetic, comparison, and logical operators. Example:

x = 5
y = 10
z = x + y  # Addition

Python Lists

Lists are mutable sequences of items. Example:

fruits = ["apple", "banana", "cherry"]

Python Tuples

Tuples are immutable sequences. Example:

coordinates = (10, 20)

Python Sets

Sets are unordered collections of unique items. Example:

fruits = {"apple", "banana", "cherry"}

Python Dictionaries

Dictionaries store key-value pairs. Example:

person = {"name": "Alice", "age": 30}

Python If...Else

The if-else statement allows decision-making. Example:

age = 18
if age >= 18:
    print("Adult")
else:
    print("Minor")

Python While Loops

While loops repeat as long as a condition is true. Example:

x = 0
while x < 5:
    print(x)
    x += 1

Python For Loops

For loops iterate over a sequence. Example:

for x in range(5):
    print(x)

Python Functions

Functions are blocks of code that are executed when called. Example:

def greet(name):
    print("Hello, " + name)
greet("Alice")

Python Lambda

Lambda functions are small anonymous functions. Example:

multiply = lambda x, y: x * y
print(multiply(2, 3))

Python Arrays

Python arrays are used to store multiple items in a single variable. Example:

import array
arr = array.array('i', [1, 2, 3, 4])

Python Classes

Classes are blueprints for creating objects. Example:

class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
car = Car("Toyota", "Camry")

Python Inheritance

Inheritance allows a class to inherit attributes and methods from another class. Example:

class ElectricCar(Car):
    def __init__(self, brand
    self.brand = brand
    self.model = model
    self.battery_size = 75
def describe(self):
    print(f"{self.brand} {self.model} with a {self.battery_size}-kWh battery")
electric_car = ElectricCar("Tesla", "Model S")
electric_car.describe()

Python Iterators

Iterators allow iterating over a sequence, like a list. Example:

nums = [1, 2, 3]
it = iter(nums)
print(next(it))  # 1

Python Polymorphism

Polymorphism allows different classes to have methods with the same name. Example:

class Animal:
def sound(self):
    pass
class Dog(Animal):
def sound(self):
    return "Woof"
class Cat(Animal):
def sound(self):
    return "Meow"
dog = Dog()
cat = Cat()
print(dog.sound())  # Woof
print(cat.sound())  # Meow

Python Scope

Scope determines the visibility of variables. Example:

x = 10  # Global scope
def func():
x = 20  # Local scope
print(x)
func()
print(x)  # Global variable

Python Modules

Modules allow you to group functions, classes, and variables. Example:

import math
print(math.sqrt(16))  # 4.0

Python Dates

Python provides the datetime module to handle dates. Example:

import datetime
today = datetime.date.today()
print(today)

Python Math

Python has a math module that provides mathematical functions. Example:

import math
print(math.pi)  # 3.141592653589793

Python JSON

Python provides a json module to handle JSON data. Example:

import json
person = {"name": "Alice", "age": 30}
json_data = json.dumps(person)
print(json_data)

Python RegEx

Python provides the re module for working with regular expressions. Example:

import re
pattern = r"\d+"
text = "There are 123 apples."
result = re.findall(pattern, text)
print(result)  # ['123']

Python PIP

PIP is the package installer for Python. Example:

pip install numpy

Python Try...Except

Try...Except handles exceptions in Python. Example:

try:
x = 10 / 0
except ZeroDivisionError:
print("Cannot divide by zero!")

Python User Input

The input function allows user input. Example:

name = input("Enter your name: ")
print(f"Hello, {name}!")

Python String Formatting

String formatting allows inserting values into strings. Example:

name = "Alice"
greeting = f"Hello, {name}!"
print(greeting)

File Handling

Python allows reading, writing, and deleting files. Example:

# Reading a file
with open('file.txt', 'r') as file:
print(file.read())

# Writing to a file
with open('file.txt', 'w') as file:
file.write("Hello, Python!")

# Deleting a file
import os
os.remove('file.txt')

Python MySQL

Python can interact with MySQL databases. Example:

import mysql.connector
conn = mysql.connector.connect(user='root', password='password', host='localhost', database='test')
cursor = conn.cursor()
cursor.execute("SELECT * FROM users")
for row in cursor.fetchall():
print(row)

Python MongoDB

Python interacts with MongoDB through the pymongo module. Example:

import pymongo
client = pymongo.MongoClient("mongodb://localhost:27017/")
db = client["mydatabase"]
collection = db["users"]
collection.insert_one({"name": "Alice", "age": 30})
