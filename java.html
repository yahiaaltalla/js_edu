Java HOME

Welcome to the Java tutorial! This guide will cover various topics related to Java programming.

Java Intro

Java is a high-level, class-based, object-oriented programming language. It is designed to have as few implementation dependencies as possible.

Java Get Started

To get started with Java, you'll need to install the Java Development Kit (JDK). After installation, you can write and run Java programs using a text editor and command line tools.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Syntax

Java syntax is the set of rules that defines the combinations of symbols that are considered to be correctly structured Java programs. Example:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

Java Output

Java programs output information using the `System.out.println()` method. Example:

System.out.println("This is a Java program output");

Java Comments

In Java, comments are used to explain code. Comments are ignored by the compiler. Example:

// This is a single-line comment

/*
This is a multi-line comment
*/

Java Variables

Variables are containers for storing data values. Example:

int x = 10;
String name = "Alice";

Java Data Types

Java has many data types, including int, double, boolean, etc. Example:

int age = 25;
boolean isStudent = true;
double price = 19.99;

Java Type Casting

Type casting is converting one type to another. Example:

int x = 10;
double y = (double) x;  // Casting int to double

Java Operators

Java has various operators like arithmetic, relational, and logical operators. Example:

int a = 5, b = 10;
int sum = a + b;  // Arithmetic operator
boolean result = a < b;  // Relational operator

Java Strings

Strings are objects that represent sequences of characters. Example:

String greeting = "Hello, " + "Java!";

Java Math

The Math class in Java provides mathematical operations. Example:

double pi = Math.PI;
int max = Math.max(10, 20);

Java Booleans

Booleans represent true/false values. Example:

boolean isAdult = true;
boolean hasPermission = false;

Java If...Else

The `if...else` statement is used for decision-making. Example:

if (age >= 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}

Java Switch

The `switch` statement is used to select one of many code blocks to be executed. Example:

int day = 3;
switch (day) {
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break;
    default: System.out.println("Invalid day");
}

Java While Loop

The `while` loop repeats a statement or block of statements as long as a given condition is true. Example:

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Java For Loop

The `for` loop is used to iterate over a range or array. Example:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Java Break/Continue

The `break` statement exits the loop, while `continue` skips the current iteration. Example:

for (int i = 0; i < 5; i++) {
    if (i == 3) break;
    System.out.println(i);
}

Java Arrays

Arrays are used to store multiple values in a single variable. Example:

int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]);  // Output: 1

Java Methods

Methods are blocks of code that perform a specific task. Example:

public static void greet() {
    System.out.println("Hello!");
}
greet();  // Calling the method

Java Method Parameters

Methods can accept parameters. Example:

public static void greet(String name) {
    System.out.println("Hello, " + name);
}
greet("Alice");  // Calling the method with a parameter

Java Method Overloading

Method overloading allows methods to have the same name but different parameters. Example:

public static void greet() {
    System.out.println("Hello!");
}
public static void greet(String name) {
    System.out.println("Hello, " + name);
}
greet();  // Calling the overloaded method
greet("Alice");

Java Scope

Scope refers to the visibility of variables. Example:

int x = 10;  // Global scope
public static void test() {
    int y = 20;  // Local scope
}

Java Recursion

Recursion is when a method calls itself. Example:

public static int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}
System.out.println(factorial(5));  // Output: 120

Java Classes

In Java, a class is a blueprint for creating objects. Example:

class Car {
    String model;
    String brand;
}

Java OOP

Java follows Object-Oriented Programming (OOP) principles like inheritance, polymorphism, encapsulation, and abstraction.

Java Classes/Objects

Classes define the properties and behaviors of objects. Example:

class Car {
    String model;
    String brand;

    void start() {
        System.out.println("Car is starting");
    }
}

Car myCar = new Car();
myCar.start();

Java Class Attributes

Class attributes (fields) define the properties of objects. Example:

class Car {
    String model;
    String brand;
    int year;
}

Java Class Methods

Methods define the behaviors of a class. Example:

class Car {
    String model;
    String brand;

    void drive() {
        System.out.println("Driving the " + model);
    }
}

Java Constructors

Constructors initialize objects. Example:

class Car {
    String model;

    Car(String model) {
        this.model = model;
    }
}

Car myCar = new Car("Toyota");

Java Modifiers

Java has various access modifiers like public, private, and protected. Example:

public class Car {
    private String model;
}

Java Encapsulation

Encapsulation is the concept of wrapping data and methods into a single unit. Example:

class Car {
    private String model;

    public String getModel() {
        return model;
    }

    public void setModel(String model) {
        this.model = model;
    }
}

Java Packages / API

Java packages are used to group related classes and interfaces. Example:

import java.util.Scanner;

public class Test {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
    }
}

Java Inheritance

Inheritance allows a class to inherit attributes and methods from another class. Example:

class Animal {
    void makeSound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    void makeSound() {
        System.out.println("Bark");
    }
}

Dog dog = new Dog();
dog.makeSound();  // Outputs: Bark

Java Polymorphism

Polymorphism allows methods to be used interchangeably. Example:

class Animal {
    void makeSound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    void makeSound() {
        System.out.println("Bark");
    }
}

Animal myAnimal = new Dog();
myAnimal.makeSound();  // Outputs: Bark

Java Inner Classes

Inner classes are classes defined within another class. Example:

class Outer {
    class Inner {
        void display() {
            System.out.println("Inner class");
        }
    }
}

Outer outer = new Outer();
Outer.Inner inner = outer.new Inner();
inner.display();

Java Abstraction

Abstraction hides implementation details and shows only functionality. Example:

abstract class Animal {
    abstract void sound();
}

class Dog extends Animal {
    void sound() {
        System.out.println("Bark");
    }
}

Java Interface

Interfaces are used to define a contract that implementing classes must follow. Example:

interface Animal {
    void sound();
}

class Dog implements Animal {
    public void sound() {
        System.out.println("Bark");
    }
}

Java Enums

Enums define a set of constants. Example:

enum Day { MONDAY, TUESDAY, WEDNESDAY }

Day today = Day.MONDAY;
System.out.println(today);  // Outputs: MONDAY

try {

Java RegEx

Regular expressions are used for pattern matching in strings. Example:

import java.util.regex.*;

String text = "Hello 123 World";

while (m.find()) {
    System.out.println("Found number: " + m.group());
}

MyThread t1 = new MyThread();

Java Lambda

Lambda expressions allow you to write concise code for functional interfaces. Example:

interface Greeting {
    void greet(String name);
}

public class Test {
    public static void main(String[] args) {
        Greeting greeting = (name) -> System.out.println("Hello " + name);
        greeting.greet("John");
    }
}

Java Advanced Sorting

Java provides powerful sorting mechanisms using `Comparator` for custom sorting. Example:

import java.util.*;

class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }
}

class PersonComparator implements Comparator {
    public int compare(Person p1, Person p2) {
        return Integer.compare(p1.getAge(), p2.getAge());
    }
}

public class Test {
    public static void main(String[] args) {
        List people = new ArrayList<>();
        people.add(new Person("Alice", 30));
        people.add(new Person("Bob", 25));

        Collections.sort(people, new PersonComparator());

        for (Person person : people) {
            System.out.println(person.getName() + " - " + person.getAge());
        }
    }
}
